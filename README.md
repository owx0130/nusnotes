# hack & roll 2024 submission

Backend:

1. server.js - The server is created here (Express.js). Functions to handle HTTP requests are contained here.
2. lib.js - Contains all the functions used by the server.  
3. llm-requirements.js - Contains all the imports for LangChain dependencies used.  

To run:

1. Make .env file in backend folder to store API key from OpenAI.
2. Run "npm install" in /backend and /frontend.
3. In backend folder, run "npm run backend".
4. In frontend folder, run "npm run frontend".
